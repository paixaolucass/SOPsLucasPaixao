### Encontros da Vanguarda — fluxo completo
| Encontro | Dia | Thumb | Nome vem de |
|---|---|---|---|
| Checkpoint | Toda quarta | 3 thumbs fixas prontas | Ruan ou Matheus |
| Workshop | Toda sexta | Midjourney (prompt) | Ruan ou Matheus |
| Masterclass | ~1x/mês, dia variável | Midjourney (prompt) | Anotado na plataforma |
| Feriado | — | ❌ Não tem | — |

**Ferramentas do processo:**
- **Zoom** → fornece gravação + bate papo + transcrição
- **CapCut** → corte do início do encontro (com ajuda de IA para identificar timestamp)
- **IA (Claude/ChatGPT)** → diagnóstico de início + geração do RI (Resumo Inteligente)
- **App Mapa Mental Vanguarda** → app próprio criado pelo Lucas; recebe RI e gera material de mapa mental
- **HandBrake** → compressão do vídeo (ex: 6GB → 1GB, ~1h)
- **Vimeo** → hospedagem; configurar nome + thumb + aguardar render → copiar link
- **Overpass (plataforma)** → publicar encontro: link Vimeo + RI na descrição + 3 materiais na aba de arquivos
- **WhatsApp Vanguarda** → avisar que encontro está disponível
- **Midjourney** → gerar capas/thumbs para Workshop e Masterclass

**Fluxo resumido:**
```
Zoom → coletar 3 arquivos → CapCut+IA (cortar início) → IA+MentaApp (RI + mapa mental)
→ plataforma (subir 3 materiais + RI) → HandBrake (~1h) // thumb paralelo
→ Vimeo (upload + nome + thumb + render) → plataforma (link + publicar) → WhatsApp ✅
```
